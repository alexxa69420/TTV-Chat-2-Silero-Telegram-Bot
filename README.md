# Twitch Chat to Telegram TTS Bot

Простой бот, который читает сообщения из чата Twitch голосом через [@silero_voice_bot](t.me/silero_voice_bot) для синтеза речи.

## Что это делает

Бот подключается к чату вашего Twitch-канала, ловит сообщения от зрителей, обрабатывает их и отправляет в бота для синтеза речи. Полученные голосовые сообщения воспроизводятся на вашем компьютере.

## Требования

- Операционная система: Windows 10/11 \\ Саппорт на маках и линуксах не обещается.
- Python 3.12 \\ у меня такая как минимум.
- Доступ в интернет
- Аккаунт Twitch
- Аккаунт Telegram \\ С подпиской на [@silero_voice_bot](t.me/silero_voice_bot)

## Настройка

### Получение данных для Telegram

1. Откройте сайт [my.telegram.org](https://my.telegram.org)
2. Войдите под своим аккаунтом Telegram
3. Перейдите в раздел "API development tools"
4. Заполните форму:
   - App title: любое название, например "TTS Bot"
   - Short name: короткое название, например "ttsbot"
   - URL: можно оставить пустым
   - Platform: выберите "Other"
5. Нажмите "Create application"
6. Скопируйте два значения: **App api_id** и **App api_hash**

### Получение токена для Twitch

1. Откройте сайт [twitchtokengenerator.com](https://twitchtokengenerator.com)
2. Сайт спросит выбрать Chat Bot Token или Custom Scope Token, нам нужен чат бот.
3. В поле с "Generated Tokens" скопироуйте тот что ACCESS TOKEN
4. Скопируйте токен и вставьте в код

### Первый запуск и настройка

1. Запустите бот командой:
   ```
   python app.py
   ```
2. При первом запуске бот запросит недостающие данные:
   - Введите `API_ID` и `API_HASH` от Telegram
   - Введите номер телефона привязанный к телеграм аккаунту, полученный от телеграма код, и облачный пароль при наличии. \\\ Не переживайте за свой аккаунт, код открыт для всех и любой человек может убедится в том что он не ворует аккаунты. Вы банально можете скормить код нейронке и спросить не ворует ли он ваши данные.
   - Введите имя вашего или бот аккаунта Twitch (логин)
   - Введите ACCESS токен Twitch
   - Введите название вашего канала на Twitch

3. Все данные сохранятся в файл `.env` рядом со скриптом

4. Бот автоматически создаст файл `config.json` с настройками

## Настройка config.json

Файл `config.json` содержит все настройки бота. Вы можете редактировать его в любом текстовом редакторе. Бот автоматически подхватит изменения без перезапуска.

### Структура файла

```json
{
  "voices": ["bandit", "arthas", "anubarak"],
  "blacklist_users": ["streamelements", "nightbot"],
  "blacklist_phrases": ["!command", "!discord", "https://"],
  "settings": {
    "target_bot": "silero_voice_bot",
    "response_timeout": 45,
    "audio_volume": 0.7,
    "flask_host": "127.0.0.1",
    "flask_port": 8124
  }
}
```

### Описание полей

**voices** — список префиксов голосов для синтеза речи. Бот будет случайным образом выбирать голос из этого списка для каждого сообщения. Примеры префиксов: `bandit`, `arthas`, `anubarak`, `geralt`. Полный список доступных голосов смотрите в документации бота `silero_voice_bot`.

**blacklist_users** — список имён пользователей, сообщения от которых будут игнорироваться. Добавьте сюда системных ботов (streamelements, nightbot, jeetbot) и тех зрителей, чьи сообщения не нужно озвучивать. Имена указываются в нижнем регистре, без символа @.

**blacklist_phrases** — список фраз, при наличии которых сообщение будет проигнорировано. Полезно для игнорирования команд чата, ссылок и другого служебного контента. Сообщения начинающиеся на ! автоматически игнорируются.

**settings.target_bot** — имя Telegram-бота для синтеза речи. По умолчанию `silero_voice_bot`. Не меняйте, если не уверены.

**settings.response_timeout** — время в секундах, которое бот ждёт ответа от сервиса синтеза. Если ответ не пришёл за это время, запрос считается неудачным.

**settings.audio_volume** — громкость воспроизведения. Значение от 0.1 (тихо) до 2.0 (громко). По умолчанию 0.7.

**settings.flask_host** и **settings.flask_port** — настройки внутреннего веб-сервера. Не меняйте, если не понимаете, зачем это нужно.

## Как пользоваться

1. Запустите бот файлом `start.bat`
2. Дождитесь сообщений в консоли:
   - `✅ Вход в Telegram` — подключение к Telegram успешно
   - `✅ Вход в Twitch` — подключение к чату успешно
   - `🔄 Worker запущен` — система обработки сообщений готова

3. Теперь все сообщения из чата вашего Twitch-канала будут автоматически озвучиваться

### Управление озвучкой

- Сообщения, начинающиеся с `/` или `!`, игнорируются (команды чата)
- Пользователи из `blacklist_users` не озвучиваются
- Сообщения, содержащие фразы из `blacklist_phrases`, игнорируются
- Числа автоматически преобразуются в слова (123 → "сто двадцать три")

## Файлы, которые создаёт бот

- `.env` — ваши секретные данные (API-ключи, токены). Никому не передавайте этот файл.
- `config.json` — настройки бота. Можно редактировать.
- `tts_bot.log` — файл с логами работы бота. Полезен для диагностики проблем.
- `tts_session.session` — папка с данными сессии Telegram. Если код отказывается работать - попробуйте удалить оба файла и перезайти в телеграм аккаунт.

## Частые проблемы и решения

### Бот не запускается, ошибка "ModuleNotFoundError"

Не все зависимости установлены. Выполните заново внутри папки:
```
pip install -r requirements.txt
```

### Ошибка при запуске, "API_ID и API_HASH не найдены"

Файл `.env` не создан или повреждён. Удалите его и запустите бот заново — он запросит данные повторно.

### Бот подключился к Twitch, но сообщения не озвучиваются

Проверьте:
1. Правильно ли указано название канала в `.env` (без символа #)
2. Не добавлен ли ваш никнейм в `blacklist_users`
3. Работает ли бот `silero_voice_bot` в Telegram (напишите ему вручную)

### Звук не воспроизводится

1. Убедитесь, что в системе не выключен звук
2. Проверьте, установлена ли библиотека `pygame` или `playsound`
3. Попробуйте изменить `audio_volume` в `config.json`

### Бот "завис" и перестал реагировать на сообщения

Нажмите `Ctrl+C` в окне консоли для корректного завершения работы бота, затем запустите его заново.

### Ошибка "Mystem not found" при первом запуске

Библиотека Mystem скачивает дополнительные файлы при первом запуске. Дождитесь завершения загрузки. Если ошибка повторяется, проверьте подключение к интернету.

## Как остановить бота

Нажмите `Ctrl+C` в окне консоли, где запущен бот. Бот корректно завершит работу и отключится от всех сервисов.
